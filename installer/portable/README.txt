PNG to ICO Converter - Portable Version

This is a portable version of the PNG to ICO Converter that requires no installation.

Requirements:
- Python 3.8 or higher
- Pillow (PIL) library
- tkinter (usually included with Python)

Usage:
1. Run run.bat to start the application
2. Or run: python main.py
3. Select your PNG files
4. Choose output location
5. Convert!

The config.ini file contains application settings and can be edited with any text editor.

Features:
- Convert PNG images to ICO format
- Support for multiple icon sizes
- Batch processing
- Preview pane for image selection
- Configurable settings
